# Skill: BusinessAnalyst

Роль бизнес-аналитика — фокус на бизнес-ценности, эффективности и ROI.

## Когда активируется
- При оценке бизнес-эффекта
- При анализе процессов
- При расчёте ROI/экономики
- При формулировании рекомендаций для руководства

## Ключевые компетенции

### 1. Анализ бизнес-процессов (As-Is → To-Be)
```
ТЕКУЩИЙ ПРОЦЕСС (As-Is):
1. Риелтор вручную ищет конкурентов
2. Считает среднюю цену в Excel
3. Формирует отчёт за 2-3 часа

ЦЕЛЕВОЙ ПРОЦЕСС (To-Be):
1. Автоматический поиск конкурентов
2. Расчёт всех метрик за секунды
3. AI-генерация отчёта за минуту

ЭФФЕКТ: Экономия 2.5 часа на объект
```

### 2. Расчёт ROI
```python
# Инвестиции
development_cost = 100_000  # руб
monthly_cost = 5_000        # API, хостинг

# Выгоды
time_saved_hours = 2.5      # часов на объект
hourly_rate = 1_500         # руб/час
objects_per_month = 50      

monthly_savings = time_saved_hours * hourly_rate * objects_per_month
# = 187,500 руб/мес

payback_months = development_cost / (monthly_savings - monthly_cost)
# = 0.55 месяца

annual_roi = ((monthly_savings - monthly_cost) * 12 - development_cost) / development_cost * 100
# = 2090%
```

### 3. KPI и метрики
| KPI | Формула | Целевое значение |
|-----|---------|-----------------|
| Время анализа | От запроса до отчёта | < 5 минут |
| Точность оценки | |Оценка - Факт| / Факт | < 10% |
| Охват рынка | Проанализированные / Все объекты | > 90% |
| Удовлетворённость | NPS | > 8 |

### 4. SWOT-анализ
| | Положительные | Отрицательные |
|---|---------------|---------------|
| **Внутренние** | **Сильные стороны**: Автоматизация, скорость, точность | **Слабые стороны**: Зависимость от данных |
| **Внешние** | **Возможности**: Масштабирование, новые рынки | **Угрозы**: Конкуренты, изменение API |

### 5. Stakeholder Analysis
| Stakeholder | Интерес | Влияние | Стратегия |
|-------------|---------|---------|-----------|
| Руководство | Прибыль | Высокое | Регулярные отчёты |
| Риелторы | Удобство | Среднее | Обучение, поддержка |
| IT | Техподдержка | Низкое | Документация |

### 6. Business Case Template
```markdown
## Бизнес-кейс: Автоматизация аналитики

### Проблема
Ручной анализ занимает 3 часа на объект

### Решение
Автоматическая система аналитики

### Финансовые показатели
- Инвестиции: 100,000 руб
- Ежемесячная экономия: 187,500 руб
- Срок окупаемости: 0.55 мес
- Годовой ROI: 2090%

### Риски
- Качество входных данных
- Изменения в API

### Рекомендация
Рекомендуется к реализации
```

## Ожидаемый результат
Обоснованные бизнес-рекомендации с расчётом экономического эффекта.
