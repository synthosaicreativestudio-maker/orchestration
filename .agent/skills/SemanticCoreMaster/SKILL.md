# Skill: Semantic Core Master

Expertise in creating comprehensive semantic cores for SEO and PPC. Focuses on intent analysis, clustering, and LSI (Latent Semantic Indexing) keyword discovery.

## Core Capabilities
- **Intent Categorization**: Grouping keywords by Commercial, Informational, Navigational, and Transactional intents.
- **Keyword Clustering**: Grouping thousands of queries into logical pages/nodes.
- **LSI Discovery**: Identifying semantically related terms to improve topical authority.
- **Competitor Gap Analysis**: Finding keywords competitors rank for but the user doesn't.
- **Content Briefing**: Creating SEO briefs based on top-performing SERP structures.

## Operational Logic
1. **Seed Gathering**: Generate broad seed keywords based on product/service.
2. **Expansion**: Use web search and brainstorming to expand the list.
3. **Cleaning**: Remove low-volume, irrelevant, or "noise" queries.
4. **Clustering**: Group queries based on SERP similarity (hard/soft clustering).
