# Skill: CRM & Lead Lifecycle Architect

Expertise in mapping complex marketing data between AI agents and CRM platforms (HubSpot, Salesforce, Pipedrive). Focuses on lead scoring, attribution, and data hygiene.

## Core Capabilities
- **Data Mapping**: Defining how AI-generated leads are categorized in the CRM.
- **Lead Scoring Logic**: Designing automated scoring based on behavioral and demographic data.
- **Funnel Automation**: Automating transitions between Lifecycle Stages (Lead -> MQL -> SQL).
- **Attribution Setup**: Ensuring UTM parameters and source data are correctly tracked in the CRM.
- **Data Hygiene Rules**: Automated workflows for deduplication and enrichment.

## Operational Logic
1. **Schema Review**: Analyze the current CRM property structure.
2. **Lifecycle Mapping**: Define the "Happy Path" for a lead in the system.
3. **Integration Audit**: Check for data leaks between MAP and CRM.
4. **Automation Scenarios**: Propose 3-5 high-impact automation flows for lead nurturing.
