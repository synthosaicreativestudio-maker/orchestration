# Growth & SEO Engineering Skill

This skill ensures the product is built to sell itself. Code is marketing.

## 1. Core Principles
- **Speed is a Feature:** Google penalizes slow sites. Target Lighthouse Score > 95.
- **Shareability:** Every page must have perfect OpenGraph (og:image, og:title) tags for Telegram/WhatsApp previews.
- **Conversion Centric:** Every page needs a clear, singular Call to Action (CTA).
- **Measurement:** "If you can't measure it, you can't improve it." Analytics events for every key action.

## 2. Technical Implementation
- **SSR/SSG:** Use Server-Side Rendering (Next.js) for critical landing pages for SEO.
- **Semantic HTML:** Use `<article>`, `<header>`, `<main>` for accessibility and SEO bots.
- **Sitemap & Robots:** auto-generate `sitemap.xml` and `robots.txt`.
- **Structured Data:** Inject JSON-LD (Schema.org) to help Google understand the product.

## 3. Pre-Launch Checklist
1.  [ ] Lighthouse Performance > 90?
2.  [ ] Lighthouse SEO > 95?
3.  [ ] Twitter Card / OG Image validates correctly?
4.  [ ] Analytics script is async and non-blocking?
