# Skill: ReportGeneration

Навык создания структурированных аналитических отчётов.

## Когда активируется
- После завершения анализа
- По запросу "Сформируй отчёт", "Подготовь резюме"
- При необходимости презентовать результаты

## Структура отчёта

### 1. Executive Summary (Краткое резюме)
- 3-5 ключевых выводов
- Главная рекомендация
- Критические метрики

### 2. Основная часть
- Описание методологии
- Детальный анализ
- Таблицы и графики

### 3. Выводы и рекомендации
- Конкретные действия
- Риски и ограничения
- Следующие шаги

## Шаблон отчёта

```markdown
# Аналитический отчёт: [Название]

**Дата**: [DD.MM.YYYY]  
**Аналитик**: AI Assistant

---

## Краткое резюме

> **Ключевой вывод**: [Одно предложение]

| Метрика | Значение |
|---------|----------|
| [Название 1] | [Значение 1] |
| [Название 2] | [Значение 2] |

**Рекомендация**: [Одно предложение]

---

## Методология

[Описание использованных методов]

---

## Результаты анализа

### [Раздел 1]
[Текст с таблицами и графиками]

### [Раздел 2]
[Текст с таблицами и графиками]

---

## Выводы

1. [Вывод 1]
2. [Вывод 2]
3. [Вывод 3]

## Рекомендации

1. **[Действие 1]** — [обоснование]
2. **[Действие 2]** — [обоснование]

---

*Отчёт подготовлен автоматически*
```

## Правила делового стиля
1. **Конкретность** — цифры вместо "много/мало"
2. **Краткость** — убирать лишние слова
3. **Структурность** — заголовки, списки, таблицы
4. **Объективность** — факты, не мнения
5. **Actionable** — конкретные рекомендации

## Инструменты
- Markdown — основной формат
- Таблицы — для сравнений
- Встроенные графики — для наглядности

## Ожидаемый результат
Структурированный отчёт в деловом стиле, понятный заказчику без технического бэкграунда.
